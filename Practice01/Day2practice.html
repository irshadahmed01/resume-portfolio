Formulas Used

IF Formula: =IF(C3>5000, "High", "Low") in D3:D102 to categorize investments as "HIGH" or "LOW" based on Current Value.

VLOOKUP: =VLOOKUP("401K", B3:C52, 2, FALSE) to look up the Current Value of specific investments (e.g., 401K: $6,597).

COUNTIF: =COUNTIF(D3:D102, "HIGH") in A106 to count high-value investments (51).

SUM: =SUM(A3:A102) in A103 for Total Invested ($53,580), =SUM(C3:C102) in C103 for Total Current Value ($519,700).

AVERAGE: =AVERAGE(A3:A102) in A104 for Average Invested ($535.80), =AVERAGE(C3:C102) in C104 for Average Current Value ($5,197).

SUMIF: =SUMIF(D3:D102, "HIGH", C3:C102) in B112 for Total High-Value Current Value ($385,850), =SUMIF(D3:D102, "LOW", C3:C102) in B114 for Total Low-Value Current Value ($133,850).

AVERAGEIF: =AVERAGEIF(D3:D102, "HIGH", C3:C102) in B113 for Average High-Value Current Value ($7,565.69), =AVERAGEIF(D3:D102, "LOW", C3:C102) in B115 for Average Low-Value Current Value ($2,731.63).
