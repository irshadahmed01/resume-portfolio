Day 1: Basics of Google Sheets

Objective

Learn the fundamentals of Google Sheets to create and summarize a 50-row investment dataset.

Key Formulas

  • SUM: `=SUM(A1:A50)` - Totals a range of values (e.g., Dollars Invested).
  • AVERAGE: `=AVERAGE(B1:B50)` - Calculates the mean (e.g., Current Value).
  • COUNT: `=COUNT(C1:C50)` - Counts non-empty cells (e.g., Investment Names).

Step-by-Step Process

  1. Create a new Google Sheet and input a 50-row dataset with columns: Investment Name, Dollars Invested, Current Value.
  2. Use `SORT` (e.g., `=SORT(A1:C50, 2, TRUE)`) to order by Dollars Invested.
  3. Apply `FILTER` (e.g., `=FILTER(A1:C50, C1:C50>5000)`) to show high-value investments.
  4. Calculate summary statistics using SUM, AVERAGE, and COUNT in a separate summary row.

Key Takeaway

Mastered basic data entry, sorting, filtering, and summary calculations, laying the foundation for data analysis.

Day 2: Intermediate Google Sheets Skills

Objective

Expand to 100 rows, clean data, and use intermediate functions.

Key Formulas

  • IF: `=IF(A1>100, "High", "Low")` - Categorizes investments.
  • VLOOKUP: `=VLOOKUP("NVIDIA", A1:C100, 3, FALSE)` - Looks up Current Value.
  • SUMIF: `=SUMIF(A1:A100, ">5000", B1:B100)` - Sums based on a condition.
  • AVERAGEIF: `=AVERAGEIF(C1:C100, ">7000", C1:C100)` - Averages high values.

Step-by-Step Process

  1. Expand the dataset to 100 rows and remove duplicates with `Data > Remove Duplicates`.
  2. Use IF to categorize investments by Dollars Invested.
  3. Apply VLOOKUP to find specific investment details.
  4. Use SUMIF and AVERAGEIF for conditional summaries.
  5. Add conditional formatting (e.g., `Format > Conditional Formatting` for values > $5,000).

Key Takeaway

Improved data cleaning and conditional analysis skills, enhancing portfolio insights.

Day 3: Advanced Google Sheets Skills

Objective

Visualize data and perform advanced analysis.

Key Formulas

  • Growth %: `=((C1-B1)/B1)*100` - Calculates percentage growth.
  • QUARTILE: `=QUARTILE(C1:C100, 1)` - Identifies outliers (Q1).

Step-by-Step Process

  1. Create a pie chart: `Insert > Chart > Pie Chart` for High vs. Low values.
  2. Build a pivot table: `Insert > Pivot Table` to summarize by Investment Type.
  3. Calculate Growth % for each row.
  4. Use QUARTILE to detect outliers and analyze their impact.
  5. Write an analysis noting volatility and diversification needs.

Key Takeaway

Gained skills in visualization and outlier detection, critical for data-driven decisions.

Day 5: Introduction to Google Apps Script for Automation

Objective

Automate tasks using Google Apps Script.

Key Code

function calculateMetrics() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var sum = 0;
  for (var i = 1; i < data.length; i++) {
    sum += data[i][1]; // Sum Dollars Invested
  }
  sheet.getRange("D1").setValue("Total Invested: $" + sum);
}

Step-by-Step Process

Day 8: Advanced Machine Learning with Python

Objective

Improve predictions with polynomial regression.

Key Code

from sklearn.preprocessing import PolynomialFeatures
poly = PolynomialFeatures(degree=2)
X_poly = poly.fit_transform(X)
poly_model = LinearRegression().fit(X_poly, y)
print(poly_model.predict(poly.transform([[1000]])))  # Predicts $1838.65

Step-by-Step Process

  1. Transform data with PolynomialFeatures.
  2. Fit and evaluate the polynomial model (MSE: 438,929.92).
  3. Refine visualizations to confirm trends.
